Installation & Registration

How to install SR device

The SR Smart App will operate any SR device. Upon selecting the SR device you will have all features and functionalities for that device in the SR Smart App. The app is available for download through the App Store or Play Store on your Smartphone. Search for "SR Smart" or "SecuRemote".The term "Smartphone" is used generically and includes devices that use the same operating system (OS).

For information about installing and operating an SR device please refer to the specific SR Smart device manual that came with the device you purchased. For SR products you can download the SecuRemote User Manual from http://www.firstwatchsecurity.com/download/sr_smart_manual.pdf

Please noted the term "devices" or "SR devices" refers to SR Smart enabled devices that work with the SR Smart App. "SR" is used as an abbreviation for "SecuRemote".

How to install SR Smart app in smart phone

The SR Smart App will operate any SR device. Upon selecting the SR device you will have all features and functionality for that device in the SR Smart App. For iOS user go to "App Store" and search for keyword SR Smart or SecuRemote and click on free > install to install app in your smartphone. For Android user go to "Google Play" and search for keyword SR Smart or SecuRemote and click on install to install app in your smartphone.

Register or Add SR Device

To Operate SR Device using Smartphone, you must need to register it on SR Portal. You can register by following any of these methods.

  1. Scan QR Code

    Press on Scan QR Code option using SR Smart App. A Camera will open to scan Serial Number and Security Token from User manual or interior part of SR Device.QR Scan will read required device info successfully. Now press on "Create New Account", It will ask email id, password and user device name afterwards press on "Register SR Device". It will register SR device in your account. If you have already created your account then you can go with "Choose existing account" option. SR Smart App will guide you for the same.

  2. Enter Manually

    Enter all required information manually using SR Smart App.Enter Serial Number, Security Token and SR Device Name.Now press on "Create New Account", It will ask email id, password and user device name afterwards press on "Register SR Device". It will register SR device in your account. If you have already created your account then you can go with "Choose existing account" option. SR Smart App will guide you for the same.

  3. Using Invite Code

Invited user can download Application using link provided in email or see above section "How to install SR Smart app in smart phone" for the same. Basic Functionality

Operations on SR device

SR Smart App is designed to lock, unlock, check the status and perform many other features of SR device. SR Smart App makes life easier and secure.

  • Red Lock/close Icon: Press this icon to lock/close the device.
  • Green unlock/open Icon: Press the icon to unlock /open the device.
  • Status: Press to check the status of the device. The Status on the right top corner.

    • Operation Modes

    SR devices can operate in any one of the modes described below. Each mode offers capabilities that can match you individual needs. Refer to your SR device manual for more information.

    Local Mode (Direct) using BLE

    In this mode the communication between an SR device and a Smartphone is direct (between the two devices) using Bluetooth. Communication distance can be up to 100 feet depending on communication obstructions or the product application. This mode also supports Notification only within Bluetooth range. The first user connects to SR Device will be treated as Admin.

    Local Mode (Online Authentication & Authorization)

    In this mode the communication between an SR device and a Smartphone is direct (between the two devices) using Bluetooth but the Administration is handled by SecuRemote Online Portal (SRP) with which a Smartphone communicates automatically in the background (through its Internet connectivity capability). No LAN with Internet connectivity is required for a device.
    Setting up an SRP account is easy and free (up to a certain number of users). This mode gives the Administrator the ability to make Administrative changes directly from their Smartphone from anywhere in real time. Similarly, although user's smartphone must be in Bluetooth range with a device in order to operate it, the authentication and authorization is performed in the background by SRP. This mode offers real-time Notification when a User operates a device.

    Local Mode (Indirect) using BLE (Using BLE + Proprietary Protocol)

    This mode offers user to operate SR device when SR device is not in direct BLE range and other SR device is in BLE range. SR Smart App automatically send request to another SR device which is available in direct range. User does not have to do anything extra to operate SR device in this mode.

    Remote Mode using SR Bridge

    This mode offers all the previous benefits but it also adds the extra feature to operate SR device from remote location. SR device is connected to SRP all the time this allowing event monitoring even when user's smartphone is not present.

    Other functionality

    SR-enabled door locks include keeler and deadbolt types under the Keeler and the SecuRemote Security brands. Depending on the type, these SR locks have unique features for power management, and other automation functionality. Some of these features are described below. Refer to the specific SR device user manual for more information.

    Auto Connect

    This function allows user to connect automatically with smart deadbolt when they come back to home and it allows user to unlock door with exterior smart button. User don't need to unlock the door from smartphone.

    Autolock Activation

    The Autolock feature allows for the automatic activation of the deadbolt in the lock position after the door closes. Autolock can be activated by pressing Autolock activation switch.

    Autolock time can control by Autolock Timer option from SR Device Settings of smartphone application.

    Settings > SR Device Settings > Auto Lock Timer. The timer offers several delay choices between 2 second and 60 seconds. The counting starts after the door is closed and the latch bolt is fully depressed in the "close" position. This feature is available only in some lock models, refer to your SR device manual.

    Audio Activation

    In addition to light indicators some locks are equipped with sound or audio that announces an operation. To turn the audio on or off: Settings > SR Device Settings > Audio.

    Audit Trail

    It's an extra security feature that allows user to monitor SR device. Its display user name with date and time and type of operation user performed.

    Battery Power Information

    The power indicator on the top left of the SR Smart App Home screen shows the type of power used (Lithium or Alkaline battery pack, or AC adapter) and the remaining % of battery life remaining if a battery pack is used. When the remaining battery life drops below 20% the SR Smart App will generate a warning every time an operation is performed.

    Over The Air (Firmware Upgrade)

    User can upgrade new firmware of SR device over the air (OTA) via smartphone.

    The SecuRemote chipset is equipped with the capability to install the latest updated software this allows the device to be upgraded in the field.

    Upgrade Firmware file will be send on user's registered email address.

    Click on Settings > About SR Device > Firmware Upgrade

    Notification

    "Doors Unintentionally Left Open or Unlocked" is a notification alert which is triggered if you have left home and moved a mile away from the SR device location. This is accomplished by SR tracking both the SR device status and your approximate location by using your Smartphone's Location Services.

    The accuracy of your location depends on the type of Location Services used by your Smartphone and the speed at which you are moving away from your home/office (SR device location). When a door is left open/unlocked SRP will request your location from the SR Smart App and will trigger the notification alert once your physical distance has exceeded a mile from the device. Your exact physical distance at the time the notification alert message is shown on your Smartphone screen will depend on the GPS connectivity and speed at which you are driving away - it will typically be about a mile.

    For iPhone users, please make sure you keep your Wi-Fi always ON. For more accurate results iPhone will use public Wi-Fi networks to calculate your current location.

    The notification is very reliable as long as your Smartphone Location Services are enabled and the SR devices are powered ON.

    SR LAN Scan Timer

    This option allows user to specify the local area network scan interval time. SR Device will stop its scanning within Protocol layer/Master Device after this much of time and will go in sleep mode. This feature will help to increase battery life of your SR device by not doing unnecessary scanning.

    SR Bridge

    Two types of SR Bridge are available. One which SR Bridge and an internet router with an Ethernet port and other which has CDMA module inside SR Bridge. Both provide same functionality. User can identify type from the user kit came with it.

    SR Bridge with Ethernet LAN/ Connectivity

    The SR Bridge is a device that connects to an Ethernet LAN and provides Internet connectivity to any SR device within wireless range. The Bridge automatically discovers and establishes communication with SR devices that are within range and are registered on the user's SRP account.

    The SR Bridge can be connected to an Ethernet LAN in a variety of ways that can extend its reachability considerably. Specifically, the SR Bridge will establish Internet access by connecting to an Ethernet port of anyone of the following devices:

    1. An Ethernet router or switch.
    2. A Wi-Fi range extender (many types are available by all major manufacturers).
    3. A powerline Ethernet adapter (also known as AC wall plugged Ethernet extender/bridge). These devices transmit data over the electrical wiring of a building.
    Beyond the practical and low-cost connectivity options above, the SR Bridge brings the benefit of plug-and-connect SR devices to the Internet. The Gateway requires no network configuration; and is able to automatically initiate and maintain communication between the SR devices and SRP.The SR Bridge does not provide support for static IP address assignment and requires a DHCP-allocated IP address in order to operate. This is typically not a problem as most Internet routers provide DHCP services.

    SR Bridge with CDMA

    SR Bridge is plug-in play device. It has CDMA connectivity which allows it to communication with SRP. The Bridge automatically discovers and establishes communication with SR device that are within range and registered on user's SRP account. No additional connectivity set up is required.

    SR Bridge Installation and Registration

    1. Install SR Bridge

      First (in case of SR Bridge with Ethernet LAN), connect the supplied Ethernet cable by plugging one end of it into the RJ-45 port on the Bridge. Connect the other end of the cable into an available RJ-45 port on a suitable network device (router, switch, power line network adapter, etc.). Second (for both type of SR Bridge), provide power to the SR Bridge by connecting the supplied power adapter to an available 110V power outlet and the power jack on the SR Bridge.

    2. Log into SRP account

      Log in to your existing SRP account by visiting http://portal.securemote.com. If you have not created an SRP account yet, please see the "SecuRemote Online" section for instructions.

    3. Register and Configure SR Bridge

      Follow the below steps to register SR Bridge on SRP.

      1. Launch the SR Smart App and select Devices->Add Devices.

      2. Select any of below two options to register SR Bridge on SRP.

        1. Scan QR Code

          2. Press on Scan QR Code option using SR Smart App. A Camera will open to scan Serial Number from User manual or interior part of SR Device. QR Scan will read required device info successfully using QR Code and next screen will appear with SR device information filled automatically.

        2. Enter SR # manually

          3. Enter all required information manually using SR Smart App. It will register SR device in your account. Please see instruction from user kit came with SR Device for necessary information to be entered.

      To manage SR Bridge, follow below steps:

      1. Please login into your existing SRP account.

      2. Go to Bridges option.

      3. Click on Bridge device you have added against registered SR Bridge.

      4. Click on "Assign SecuRemote Device" option.

      5. Select SR Device from the Device Listing and add/update description.

      6. Click on "Add SR Device" to assign your device to your bridge.

      7. It will ask for final confirmation. Please click on "Confirm" button.

      8. From here user can manage SR Devices under that SR Bridge. This configuration can be changed anytime in future according to user's need.

    4. Confirm Configuration

      Verify that the Bridge's red LED light is on and steady. This indicates a successful configuration. A blinking red LED light indicates that the SR Bridge cannot connect to an SR device that has been assigned to it.LCD message for status of it. LCD message shows current state of SR Bridge.     Security and Safety Precautions

    Security Settings - Lost Smartphone

    Security settings are accessible only to the Administrator (the first person who pairs with the device or a SecuRemote Portal user who creates an Administrator password). In the hands of an unauthorized user the Administrator's Smartphone can certainly be abused to operate a device. Therefore, it is recommended that you enable your Smartphone's lock screen security feature so that it will not be operable if it is lost or stolen ( iPhone users should enable the "Passcode Lock" feature, for Android use "Screen Lock PIN" feature). In addition, if your Smartphone is lost or stolen you should register your devices as the new Administrator with a new password as soon as possible.

    Doors with SR Locks

    Operating any door remotely through the Internet inherits potential security and safety risks requiring certain precautions. If you operate an SR door lock remotely using SRP make sure the door is properly closed by the individual you are letting through otherwise if the door is left open the status notification on the SR Smart app will not be reliable (i.e. the deadbolt may be in the lock position while the door is left open). Regardless whether you are letting someone through via a remote operation or an authorized user is operating the lock directly on site you should always have that person ensure that the door is properly closed and secure.
    When the remaining battery life drops below 20% the SR Smart App will generate a warning every time an operation is performed. This will give you ample warning to replace the battery if not the electronics are programmed so that the last operation will not leave the lock unable to activate the deadbolt in a lock position.

    Garage Doors with SR-enabled Operators

    Operating garage doors remotely using SRP can create a safety hazard particularly if the garage door operator is not equipped with the proper safety devices that will stop and reverse the closing of a door in the event a person or a pet attempts to cross its path while closing. Therefore it is highly recommended that such safety devices are thoroughly tested for proper functionality as specified by the door operator's user manual. In addition, First Watch Security highly recommends that a garage door is closed at the presence of an adult and/or while the person who operates the door using SRP has an unobstructed view of the door either by physical proximity or the use of an IP camera.

    Doors Unintentionally Left Open or Unlocked

    SecuRemote uses Location Services functionality provided by the mobile operating system in order to enable SecuRemote automation features such as notification alerts if the user is driving away from a door which is left open or unlocked. Please refer to your SR device manual to take advantage of such features if available for that device.

    Disclaimer.Any person who performs remote closing of a garage door using SRP becomes solely responsible for the safe closing of the door and will hold First Watch Security harmless from any personal injury or property damage that may result from such operation.

    Installation: Connecting

    Q.1 How do I connect with an SR device?

    In order to operate an SR device you must first connect it with your Smartphone using your Smartphone's Operating System (OS). To do this, make sure your Smartphone Bluetooth is turned ON, the SR device is powered ON, and that both devices are within direct operation range. Refer to the SR device manual.

    Q.2 I am not able to Connect with an SR device. What should I do?

    1. Make sure your Smartphone Bluetooth is turned ON, the SR device is powered ON, and that both devices are within direct operation range.
    2. Turn OFF Bluetooth of your smart phone and Turn ON again.
    3. Now try to connect with SR Device again.

    Q.3 I am not able to Discover my SR device. What should I do?

    1. Make sure your Smartphone Bluetooth is turned ON, the SR device is powered ON, and that both devices are within direct operation range.
    2. Operate SR Device with Lock/Close or Unlock/Open or Status commands using Home Screen Menu.

    Q.4 How do I connect again with SR Device after Factory Reset ?

    1. Make sure your Smartphone Bluetooth is turned ON, the SR device is powered ON, and that both devices are within direct operation range.
    2. Operate SR Device with Lock/Close or Unlock/Open or Status commands using Home Screen Menu.
    3. If use is invited user then he/she will get message to contact Administrator due to Factory Reset. When Administrator will execute above two steps then invited user can operate/connect with SR Device.
    4. All the settings of SR device will set to factory default.

    Q.5 How do I connect again with SR Device after Re-Installing the App ?

    1. User has to add SR Device again with SR Smart App. Read user manual for detailing.
    2. Make sure your Smartphone Bluetooth is turned ON, the SR device is powered ON, and that both devices are within direct operation range.
    3. Operate SR Device with Lock/Close or Unlock/Open or Status commands using Home Screen Menu.

    Q.6 My iPhone shows that the Bluetooth is connected but the SR device shows that the device is not connected

    This is an issue with the Bluetooth functionality of your iPhone. Turn the Bluetooth off and then on from the Bluetooth from the Bluetooth setting screen. Now try to re-connect your Bluetooth. Please refer to your smartphone user manual for further information.

    General Operation: Settings

    Q.1 How to set the User PIN ?

    Follow the below steps to set User pin.

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Select Set User PIN option "Settings" -> "SR Smart App Settings->Set User PIN".
    3. Enter default User PIN(1234) when prompted.
    4. Enter your new User PIN, confirm PIN and press "Save" button.

    Q.2 How User PIN switch feature will work ?

    If User PIN is enabled, SR Smart will ask user pin performing any operation(Lock/Unlock/Open/Close).

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Enable User PIN switch "Settings" -> "SR Smart App Settings->User PIN".

    Q.3 What to do if a user forgets the User PIN ?

    Follow the below steps user forgets the User PIN.

    1. Delete/Remove SR Smart App from your Smartphone.
    2. Install SR Smart App from Apple or Play Store.
    3. Add SR Device using SR Smart App.Refer user manual for detailing.

    Q.4 Who can change User PIN?

    Only the user who has access to iOS/Android device and who knows the current User PIN.

    Q.5 How to set Sound Notification ?

    Follow the below steps to set Sound Notification. Sound will be played in your smart phone while connect,disconnect or failure with SR Device.

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Enabled/Disable Sound Notification "Settings" -> "SR Smart App Settings->Sound Notification".

    Q.6 How to change alias name of the SR Device ?

    Follow the below steps to change alias name of SR Device.

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Give or change name of device as per your choice "Settings" -> "SR Smart App Settings" -> "SR Device Name".
    3. Enter name as per your choice and press "Save" button.

    Q.7 How to change settings of the SR Device ?

    Follow the below steps to change settings of SR Device.

    1. Launch SR Smart App, Press Settings -> SR Device Settings option.
      2. NOTE - You must need to pair with SR Device first. It will ask admin email id and password for authentication as well.
    2. Change or Update settings of SR Device as per your choice and press "Save" button.

    Q.8 How can I see audit trail of the SR Device ?

    Follow the below steps to see audit trail of SR Device.

    1. Launch SR Smart App, Press Settings option.
      2. NOTE - You must need to pair with SR Device first.
    2. Press on Audit Trail button "Settings" -> "SR Device Audit Trail".
    3. You can see the listing of commands executed on SR Device.

    Q.9 How can I get the SR Device specific Information ?

    Follow the below steps to get SR Device specific Information.

    1. Launch SR Smart App, Press Settings option.
      2. NOTE - You must need to pair with SR Device first.
    2. Press on About SR Device "Settings" -> "About SR Device".
    3. You can see the listing of SR Device specific Information.

    Q.10 How can I delete the SR Device from my account ?

    Follow the below steps to delete the SR Device from your account.

    1. Launch SR Smart App, Press Settings option.
    2. Press on Delete SR Device "Settings" -> "Delete SR Device".
    3. You need to validate with your email id and password. After validation SR device will be removed from your account.

    Q.11 How can I login or logout from my account ?

    Follow the below steps to login or logout from your account.

    1. Launch SR Smart App, Press Settings option.
    2. Press on Login "Settings" -> "Login".
    3. You need to validate with your email id and password. You just need to press Remember Me option,SR Smart will automatically authenticate your account with SRP without asking you again admin's credentials .
    4. Press on Logout "Settings" -> "Logout".
    5. Your admin credentials will be cleared and you are logged out. It will start asking admin id and password again where it is required.

    Q.12 How to set location of my SR Device ?

    Follow the below steps to set location of your SR Device.

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Press on Set Location "Settings" -> "Set Location".
    3. This option allows the user to set location of installed, SR device. In order to receive notification about door left unlocked when user is out of range, user must set location of SR device where it is installed.
    4. Make sure you are at near or within range of your SR Device..

    Q.13 How to notification of my SR Device ?

    Follow the below steps to notification of your SR Device.

    1. Launch SR Smart App, Press Settings -> SR Smart App Settings option.
    2. Enabled switch Notification "Settings" -> "Notification".
    3. This option allows user to enable or disable notification when user goes out of range from SR device installed if user has left the door unlocked.

    Q.8 "Doors Unintentionally Left Open or Unlocked" notification

    "Doors Unintentionally Left Open or Unlocked" is a notification alert which is triggered if you have left home and moved a mile away from the SR device location. This is accomplished by SR tracking both the SR device status and your approximate location by using your Smartphone's Location Services.
    The accuracy of your location depends on the type of Location Services used by your Smartphone and the speed at which you are moving away from your home/office (SR device location). When a door is left open/unlocked SRP will request your location from the SR Smart App and will trigger the notification alert once your physical distance has exceeded a mile from the device. Your exact physical distance at the time the notification alert message is shown on your Smartphone screen will depend on the GPS connectivity and speed at which you are driving away - it will typically be about a mile.
    For iPhone users, please make sure you keep your Wi-Fi always ON. For more accurate results iPhone will use public Wi-Fi networks to calculate your current location.
    The notification is very reliable as long as your Smartphone Location Services are enabled and the SR devices are powered ON.

    Q.9 Enabling the "Doors Unintentionally Left Open or Unlocked" notification

    Once you have registered an SR Bridge with SRP, the SR Smart App will automatically detect the SR Bridge and it will enable this notification functionality for SR devices that support it (SR Locks and Garage Door devices) provided that you have enabled your Smartphone's Location Services and have authorized the SR Smart App to use them.

    For iPhone:
    1. Make sure you have enabled Location Services and had given permission to the SR Smart App.
    2. You can enable Location Services using iOS Settings>Privacy>Location Services.
    For Android:
    1. Make sure you have enabled GPS/Network Provider Enabled.
    2. You can enable GPS/Network Provider using Settings>Location Services.
    To mark the SR device location:
    1. If you want to manually enable the "Doors Unintentionally Left Open or Unlocked" notification, Select "Settings-> SR Device Settings->Notification->Set Location" and make sure you are within range of the SR device.

    6. Select the registered SR Bridge device on SRP.
    7. Go to "Bridges" option.
    8. Click on Bridge device you have added against registered SR Bridge.

    GLOSSARY

    SecuRemote® (SR). An innovative technology platform that provides wireless access and control of SR-enabled locks and devices using Smartphones. With the aid of the SR Smart App, a Smartphone can control, operate, and monitor any SR device or product directly or remotely over the Internet via an IP Gateway. SR provides for individualized control and authorization ensuring the highest possible security. In addition, by leveraging the ubiquity of Smartphones, SR provides new levels of convenience to the owner (e.g., proximity-based detection and operation).

    SR Smart App.A software application designed to run on Smartphones,tablet computers and other mobile devices.Also referred to as mobile application (or mobile app).See http://en.wikipedia.org/wiki/Mobile_app.

    Smartphone.A mobile phone or device with capabilities that include wireless personal area network (WPAN) communication using Bluetooth, wireless local area network (WLAN) communication using Wi-Fi, Internet access, web browsing, email, text messaging, and GPS services. The term "Smartphone" is used generically and includes mobile devices that use the same operating system (OS). Refer to the SR device manual for supported OS platforms.

    Bluetooth. A wireless technology standard for exchanging data over short distances. See Bluetooth. Bluetooth Low Energy (BLE) is a new protocol released for low power wireless embedded devices that can operate on a single coin-cell battery from months to years, such as Wireless Sensor Network, watches, and sports equipment.

    Wi-Fi. A technology that allows an electronic device to exchange data wirelessly over a computer network, including high-speed Internet connections. See Wi-Fi.

    SR Keyfob. A small electronic device that controls SR devices wirelessly using Bluetooth. The communication includes secure authentication and authorization so that the fob acts as an electronic key to operate an SR device. The SR Smart App can setup one or more fobs to operate any SR devices.

    SR Device. The generic term for any consumer product or electronic device that is SR-enabled, meaning it uses SR technology, such as various types of door and utility locks, door operators, safes, etc.

    Factory Reset. Performed via a push button on the SR device. This procedure clears the device's memory, effectively restoring it to factory conditions, and allowing it to be configured for use. However, if a device is associated with an SRP account, a factory reset will not affect its operation. The device must be removed from the SRP account in order to be operated using direct communication. See Operation Modes below.

    SR Bridge. A device that connects to an Ethernet LAN and provides Internet connectivity to SR devices wirelessly. The Bridge automatically discovers and establishes communication with up to 6 SR devices that are within range and which are registered on the user's SRP account. The Bridge can be connected to an Ethernet router, switch, Wi-Fi range extender, or a universal powerline internet adapter via an RJ-45 Ethernet port.

    SecuRemote Portal (SRP). A secure, web-based access control system that provides centralized administration and management over multiple SR devices. SRP increases security and convenience by providing centralized management of access privileges (users, schedules, devices, etc.). The SRP powerful real-time notification features and auditing tools bring the ultimate peace-of-mind.

    Automation Control Functions.SecuRemote Technology brings powerful automation control functionality for Administration, Operation, and Notification.

    Administration. Setting rules for users and devices (who, when, what, how, under what conditions can access, operate or control a device).
    Operation. Interacting with a device to activate functional actions (based on what actions are available for a user, when, under what conditions).
    Notification. Monitoring and notification alerts based on user requests and/or operation events or conditions (what event and under what conditions, who needs the notification, how should the notification be delivered, when).

    Administrator (or Admin user). The user who has complete administrative control of all the device and security settings including the ability to add/delete other users. The Administrator can authorize another user to also assume complete administrative control as another Administrator. Therefore, an SR device or an SRP account can have multiple Administrators, also referred to as Admin users. Specifically:
    In Direct operation mode the Administrator of an SR device is the person who pairs with it first, either upon purchase or upon a factory reset of that device.
    In Online mode, the SRP Administrator is the person who creates an SRP account by providing their email address and creating a password. The SRP Administrator assumes control of an SR device by registering the device's Serial Number (SN) on the SRP account.

    Operation Modes.
    SR devices can operate in any one of the modes described below. Each mode offers capabilities that can meet different user needs. Refer to your SR device manual for more information.

    Local Mode (Direct) using BLE. In this mode the communication between an SR device and a Smartphone is direct (between the two devices) using Bluetooth. Communication distance can be up to 100 feet depending on communication obstructions or the product application. This mode also supports Administration (Users and Rules are stored on the device itself), and Notification only within Bluetooth range.The first user connects to SR Device will be treated as Admin.
    Local Mode (Online Authentication & Authorization). In this mode the communication between an SR device and a Smartphone is direct (between the two devices) using Bluetooth but the Administration is handled by SecuRemote Online Portal(SRP) with which a Smartphone communicates automatically in the background (through its Internet connectivity capability). No LAN with Internet connectivity is required for a device.
    Setting up an SRP account is easy and free (up to a certain number of users). This mode gives the Administrator the ability to make Administrative changes directly from their Smartphone from anywhere in real time. Similarly, although user's Smartphones must be in Bluetooth range with a device in order to operate it, the authentication and authorization is performed in the background by SRP. This mode offers real-time Notification when a User operates a device.
    Local Mode (Indirect) using BLE (Using BLE + Proprietary Protocol). When SR Device is not available in BLE range and user wants to operate in local mode, he/she can operate by sending request to other or near by SR Device using Proprietary Protocol. Remote Mode using SR Bridge. If SR Device is not in BLE range or User is at another location then using this operation mode, user can operate his/her SR Device using SR Bridge via Internet Connectivity and Proprietary Protocol. SRP will be used for web API.

    Direct Operation Range. Refers to the wireless communication range between a Smartphone and an SR device or between two SR devices.This range depends on the specific SR device and its range settings, the Smartphone make/model, and physical obstructions. For most applications and devices the maximum range is 100 feet. For some applications the maximum range can be considerably higher, refer to the specific SR device manual.

    Notification. Under Direct operation the SR Smart App can send email notifications to the device Administrator whenever an SR device is operated. When operating in Online mode notifications are handled by SRP according to the configured notification rules on the SRP account. Upon authorizing a user to operate a device the Administrator can opt for an automatic notification to that effect. This notification is sent to the user via SMS.

    Remote Proximity. The physical distance between a Smartphone (or SR fob) and an SR device that triggers an action/operation or notification event automatically. Depending on the SR Smart App the remote proximity range is adjustable. The remote proximity feature is available for SR devices and Smartphones equipped with BLE such as the iPhone OS 4. For such devices the SR Smart App allows the user to set their own remote proximity parameters (range, devices, users, and functionality adjunct or mutually exclusive to the manual operation using the app).

    Latency. The amount of time it takes for an SR device to operate or to respond to an SR Smart app command. Latency depends on the various wireless mobile data that intervene between a Smartphone and an Internet application, and SRP server response time for authentication and authorization. Typically, for direct Bluetooth communication (device-to-device) latency is less than 1 second. For SRP communication latency can vary from 2-5 seconds. Communication through SR Bridges latency can vary between 8 and 10 seconds.

    Location Services. Location Services allows location-based apps and websites (including SR Smart app) to use information from cellular, Wi-Fi, and Global Positioning System (GPS) networks in order to determine the user's approximate location. See http://support.apple.com/kb/HT5467.
    SecuRemote uses Location Services functionality provided by the mobile operating system in order to enable SecuRemote automation features such as notification alerts if the user is driving away from a door which is left open or unlocked.

    Communication Security. SR provides for end-to-end security of communication paths through the use of 128-bit encryption and patent-pending communication security methods. All communication between the Smartphone and SR devices utilizes Bluetooth's 128-bit E0 encryption. Likewise, all communication between the Smartphone and SRP utilizes 128-bit SSL encryption. Furthermore, sensitive data, such as the "digital key" that is issued to the device by SRP to authorize access, is further protected using 128-bit AES encryption. More security related features are described below.

    Secure Key (aka Digital Key). A unique, digital message sent from SRP to an SR Device that instructs it to operate. This key is issued by SRP when a request for operation (e.g., a user attempts to open a door secured by an SR-enabled lock that is associated with an SRP account) is received and satisfies the authentication and security constraints defined on the SRP account. The key effectively notifies the device that it is safe to operate. It is a time-sensitive message encrypted using 128-bit AES encryption. In addition, the AES encryption is performed using an encryption key that is unique to the device being operated.

    User PIN. A security feature that allows a user to create a PIN the use of which becomes necessary in order to operate an SR device (i.e. every time the user will attempt to operate the device she/he will be prompted to enter the PIN). This feature does not replace the SR security layers (see Communication Security) but offered as additional user-defined security. Depending on the application some SR devices offer this feature only as Administrator-defined security (the Administrator can impose this feature on specific users as part of their access privileges).

    Authentication. Verification of the identity of the individual attempting to operate an SR device. Under direct operation an SR device validates the identity of the individual by comparing the phone number communicated to it by the SR Smart app to the list of valid users stored within the device's memory. Conversely, when an SR device is operated in Online mode, the authentication is performed by SRP. SRP validates the identity of the individual by comparing their phone number and security token (see "Security Token") against the list of valid users on the SRP account.

    Security Token. A random series of bytes generated by the SR Smart platform at the time of manufacturing SR Device. When SRP communicates with SR Smart App for the first time it stores this "token" and requires that all further communication contain the same token. The use of a security token prevents a type of "spoofing attack" where a second phone can be programmed with the same phone number in order to masquerade as another individual. The only way to obtain the security token is to physically gain access to the individual's Smartphone and reverse-engineer the memory contents to extract the token. In the case of having been issued a replacement phone, or simply re-installing the SR Smart App on the same Smartphone, the security token can be cleared from SRP's database by the account Administrator through an Online process.

    Inviting a user (Invite Code). The Administrator can authorize any Smartphone user to operate an SR device. As part of this process the Administrator "adds" a user by entering the user information on the phonebook including the user's phone number and email address and then sending the user an Invitation Code.This code is generated automatically and becomes part of the security layer along with the Security Token, phone number, device Serial Number, etc.

    User Device. User Device indicates any of the mobile device. It will be either iOS or Android.

    Authorization. The process of determining whether an individual should be permitted to operate an SR device. Under Direct operation successful authentication (see "Authentication") of the user is the only prerequisite for authorization. However, when operating in Online mode, SRP requires successful authentication of the user and satisfaction of any security-related constraints (time-based access schedules, disabled devices, etc.) before the user is authorized and issued a digital key (see "Secure Key").
